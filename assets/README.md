# Assets Folder

This folder should contain the following images for the app:

- `icon.png` - App icon (1024x1024 recommended)
- `splash.png` - Splash screen image (1284x2778 for iPhone, or use a centered smaller image)
- `adaptive-icon.png` - Android adaptive icon foreground (1024x1024)

You can create these using any image editor. For a quick start, you can use solid color images or simple designs.

Recommended colors based on the app theme:
- Primary: #8B5A8C (purple)
- Background: #E8D5E0 (light pink)
